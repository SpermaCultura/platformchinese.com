Bonfiglio kid's massacre

Bonfiglio kid's massacre è un videogioco horror (voi considerate horror anche Doki Doki Literature Club) in stadio alpha programmato da Rajae Brazzers per conto di Sasà Arzanulla Production. Il videogioco è sviluppato in C# con Unity 2023 e *a detta dello sviluppatore* nella Release Candidate verrà reso Open-Source.

Come giocarlo


AVVISO: Attualmente il gioco si ritrova in fase alpha, e perciò molte funzioni previste per i futuri rilasci non sono attualmente disponibili. Quindi per favore, se ti sembra una cagata, ricordati di star valutando la prima Alpha Pubblica. Detto ciò, per giocare alla prima Alpha di Bonfiglio kid's massacre bisogna rispettare i seguenti requisiti:
Avere Windows 10 o 11 ¹ ²
Avere 7-zip installato. In caso contrario, è possibile scaricarlo da qui
Avere un computer un minimo potente

Come installarlo

per prima cosa, scaricate questo archivio compresso. Una volta scaricato dovrete estrarlo usando ³ 7-zip, e alla fine dell'estrazione, dovrete aprire "My project.exe".


Note:
¹ Attualmente Bonfiglio kid's massacre è disponibile solamente per Windows, al rilascio della Release verranno aggiunte le seguenti piattaforme: macOS (Intel ed Apple Silicon), Linux, Web e forse anche Android, iOS e tvOS.
² Attualmente Bonfiglio kid's massacre è stato provato solo su Windows 11, e di conseguenza non sappiamo se funziona anche con Windows 8.1 ed inferiori
³ Utilizzando Esplora File di Windows per estrarre il file darà errori, dato che il livello di compressione del file è impostato a 5 e non 1.
